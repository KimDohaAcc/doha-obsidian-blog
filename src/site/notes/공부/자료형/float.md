실수를 표현하기 위해 사용하는 자료형
single precision이라고도 한다

크기는 **4Byte = 32bit**로 int와 같다

##### 구성
sign(1 bit) + 지수 부분(8 bit) + 유효숫자 부분(23 bit) = 32 bit
