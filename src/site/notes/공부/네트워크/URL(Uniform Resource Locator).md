리소스의 위치를 의미한다

위치는 ==변경이 가능==하다
##### 예시


foo://example.com:8042/over/there?name=ferret#nose

###### Scheme
foo
일반적으로 프로토콜(ex. https, http, ftp) 정보가 들어간다

###### authority
example.com:8042
호스트명:포트번호

###### path
/over/there

###### query
name=ferret

###### fragment
nose
서버로 전송되는 정보는 아니고, 내부 북마크라고 생각
