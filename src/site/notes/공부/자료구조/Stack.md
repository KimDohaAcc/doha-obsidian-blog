peek

## 스택의 활용
1. 괄호 검사 - 하나씩 쌓아서 괄호쌍을 확인
2. Function call - 호출과 복귀
3. 실행 취소(ctrl+z / ctrl + y) - z 스택과 y 스택을 만들어 활용
4. 계산기(중위, 후위, 전위 표현)
5. 단어 뒤집기
