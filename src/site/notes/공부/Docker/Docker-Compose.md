여러 개의 도커 컨테이너를 정의하고 실행하기 위한 도구

하나의 설정 파일로 여러 개의 컨테이너를 관리하고, 컨테이너 간의 네트워크 및 종속성을 설정하는 데 사용

### 주의 사항
- Container 간의 관계를 이해하고 다뤄야 한다
- 보안 때문에라도 Docker network 사용 필수다
